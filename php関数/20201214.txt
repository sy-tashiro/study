■fgets()
	引数にファイルポインタを指定し、ファイルから一行を読み込み、文字列として返します。
	fgets(ハンドル、[長さ]);
	長さを指定しない場合は改行かEOFまで読み込みます。

■fopen()
	ファイルへの接続をオープンする
	$変数名(ファイルポインタ) = fopen (“開きたいファイル名”, “オープンモード”);
	"w"　　　　　	エラーが出る	先頭
	"x"　　　　　　	失敗(エラー)する	先頭
	"c"　　　　　　	新規作成する	先頭
	"a" ※追記専用	新規作成する	末尾

■feof()
ファイルポインタがファイルの終端にあるどうかを判定します。
feof ( ハンドル );

■fclose()
ハンドルが指定しているファイルをクローズする
fclose(ハンドル);

■header()
webクライアントにCSVドキュメントを送る際は、クライアントにCSVドキュメントを送付する事を知らせるために
HTMＬヘッダには文字コードやファイルの種類などの情報が含まれており、header()関数を利用して送信することができます。

■fgetcsv()
カンマで区切りで構成されたCSVファイルの行を読み込む
$配列 =fgetcsv(ファイルポインタ、行の長さ);

■is_readable()
ファイルが読み込み可能かどうかを判定する。
is_readable(ファイル名またはファイルへのパス);

■is_writable()
書き込み可能かどうかを判定する。
is_writable(ファイル名)
存在して書き込み可能な場合trueを返します。

■file_exists()
ファイルやディレクトリが存在しているかどうかを判定する。
ファイルあるいはディレクトリへのパス。

Windows 上でネットワーク共有上のファイルを調べるには、 //computername/share/filename または \\computername\share\filename のように指定してください。

■file_get_contents()
ファイルの内容を全て文字列に読み込む

string file_get_contents ( string $パス/ファイル名 [, bool $インクルードパス = false [, $コンテキスト [, int $オフセット値 = 0 [, int $最大バイト数 ]]]] );
